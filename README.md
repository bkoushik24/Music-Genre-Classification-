# Music-Genre-Classification-
A Deep Learning project that automatically classifies music into different genres using audio signal processing and Convolutional Neural Networks (CNN). The system extracts audio features using Librosa, converts them into Mel Spectrograms, and uses a CNN model built with TensorFlow/Keras to predict the genre of the music file.
# 📌 Project Overview

Music Genre Classification is an important task in **Music Information Retrieval (MIR)** systems. Streaming platforms such as Spotify and Apple Music use similar technologies to organize songs and recommend music to users.

In this project, audio signals are converted into **Mel Spectrogram representations**, which are treated as images and fed into a **Convolutional Neural Network** for classification.

The model predicts genres such as:

- Blues
- Classical
- Country
- Disco
- HipHop
- Jazz
- Metal
- Pop
- Reggae
- Rock

---
# 🎯 Objective

- Automatically classify music tracks into predefined genres
- Extract meaningful audio features using **Librosa**
- Train a **CNN model** for spectrogram classification
- Build a **Streamlit web interface** for real-time genre prediction

# 📂 Dataset

The project uses the **GTZAN Genre Dataset**.

Dataset Details:

- 1000 audio tracks  
- 10 music genres  
- Each audio file is **30 seconds long**

Genres Included:

- Blues  
- Classical  
- Country  
- Disco  
- HipHop  
- Jazz  
- Metal  
- Pop  
- Reggae  
- Rock
